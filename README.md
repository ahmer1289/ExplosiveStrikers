# ExplosiveStrikers

**ExplosiveStrikers** is a chaotic 2-player game on a single screen where players compete to score goals while dodging and throwing explosive apples at each other! Each player has unique key bindings, making the gameplay fast, fun, and unpredictable.

## How to Play

- **Player 1 Keys:** W/A/S/D for movement, Space to throw apples  
- **Player 2 Keys:** Arrow Keys for movement, Enter to throw apples  
- Score goals while avoiding explosive apples!  
- Dodge, throw, and compete for the highest score.  

## Features

- Local 2-player competitive mode  
- Explosive apples add chaotic fun  
- Fast-paced, simple, and addictive gameplay  
- Colorful and dynamic single-screen arena
